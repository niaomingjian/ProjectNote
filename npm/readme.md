## npm-config  
Manage the npm configuration files  

+ display registry:  
$ npm config get registry  

result:  
```
https://registry.npmjs.org/
```

## npm-ls
List installed packages  

+ display the version of lodash:  
$ npm list lodash  

result:  
```
D:\blgitlab\npm-packages
`-- lodash@4.17.4
```
